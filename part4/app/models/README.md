# 🏛️ Models — part4

Data models for the HBnB application.

## Files

Python classes representing the core business entities.

## Entities

- **BaseModel** — Abstract base with UUID and timestamps
- **User** — User account (email, password, first/last name)
- **Place** — Rental property (title, description, price, location)
- **Amenity** — Property feature (name)
- **Review** — User review (text, rating)
