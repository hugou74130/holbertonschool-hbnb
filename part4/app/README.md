# 🚀 HBnB Part 4 — Full Stack Application

Complete HBnB implementation with admin features and static frontend.

## Structure

| Directory | Description |
|-----------|-------------|
| `api/` | REST API with admin endpoints |
| `models/` | SQLAlchemy ORM models |
| `persistence/` | SQLAlchemy repository |
| `services/` | Business logic facade & user repository |
| `tests/` | Comprehensive test suite |

## Key Features

- Admin privileges & protected endpoints
- File upload support (place images)
- Static HTML/CSS/JS frontend
- Database seeding with `populate.py`
- Full CRUD for all entities
