# 🎨 HBnB Static Frontend

Simple HTML/CSS/JS frontend for the HBnB application.

## Files

| File | Description |
|------|-------------|
| `index.html` | Homepage — list of places |
| `login.html` | Login page |
| `place.html` | Place detail page |
| `add_review.html` | Add review page |
| `styles.css` | Global styles |
| `scripts.js` | Frontend JavaScript |
| `images/` | Place images |

## Usage

Open `index.html` in a browser after starting the Flask server.
